A. Parity

题意：给定一个多项式，判断值的奇偶，第 $i$ 项为 $a_i \cdot b^{k-i} $。

解法一：直接模拟，保持对 $2$ 取模。
解法二：判断 $base$ 和最后一项的奇偶性。

B. Tape

题意：一根木棍，分成大小为 $1$ 的 $m$ 段区间，上有 $n$ 个裂缝，要求使用不超过 $k$ 条胶布覆盖所有裂缝。问最短使用多少胶布。

可以认为有 $n$ 个节点的图，用 $n-k$ 条边连接成 $k$ 个联通块，贪心取前 $n-k$ 个最短的区间，加上必须覆盖的 $n$ 个区间。

C. Meaningless Operations

题意：给定 $a$ 求 $f(a) = \max_{0 < b < a}{gcd(a \oplus b, a \& b)}$

对 $a$ 的二进制表示中每一位取反，使 $a \oplus b = 2 ^ n - 1$，$a \& b = 0$，使两数异或值最大，按位与值等于 $0$，使结果最大化
特例：当 $a$ 为 $2$ 的幂次减 $1$ 时，$b$ 不可取 $0$，贪心取小于 $a$ 的最大因子

p.s. 内建函数 __builtin_popcountll() 用于统计二进制表示中 $1$ 的个数
